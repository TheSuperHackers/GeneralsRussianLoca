Russian Community Language Pack for COMMAND AND CONQUER GENERALS


----------------------------------------------------------------------------------------------------------------------
--- REQUIREMENTS -----------------------------------------------------------------------------------------------------

This addon requires an installation of 'Command and Conquer Generals'

You can buy it with the 'Command & Conquer - The Ultimate Collection' bundle from

https://www.origin.com


----------------------------------------------------------------------------------------------------------------------
--- INSTALL ----------------------------------------------------------------------------------------------------------

1. Copy all .big files and Data folder into the game install directory, for example:
   > C:\Program Files (x86)\EA Games\Command and Conquer Generals
   > C:\Program Files (x86)\EA Games\Command & Conquer The First Decade\Command & Conquer(tm) Generals
   > C:\Program Files (x86)\Origin Games\Command and Conquer Generals Zero Hour\Command and Conquer Generals

2. Run Registry\ActivateRussian.reg

3. [Optional] Install bundled fonts into C:\Windows\Fonts
